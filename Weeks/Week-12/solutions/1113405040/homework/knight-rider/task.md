# Homework: Knight Rider 跑馬燈

## 目標

完成 `main.py` 中標記 `TODO` 的部分，讓 4 顆 LED 做出 Knight Rider 來回掃描效果。

## 預期行為

```
[*][ ][ ][ ]  →  [ ][*][ ][ ]  →  [ ][ ][*][ ]  →  [ ][ ][ ][*]
                                                            ↓
[*][ ][ ][ ]  ←  [ ][*][ ][ ]  ←  [ ][ ][*][ ]  ←  [ ][ ][ ][*]
```

每步間隔 0.15 秒，只有一顆 LED 亮著。

## 提示

1. `all_off()` 已寫好，直接呼叫即可。
2. 用 `leds[pos]` 存取指定位置的 LED。
3. `pos` 到達 `0` 或 `3` 時，需要改變 `direction`。
4. 修正 `pos` 的方式：反轉後再移動一步，或直接 clamp 到邊界。

## 挑戰題（選做）

- 讓速度可以調整：按某個條件加速或減速。
- 同時讓 2 顆 LED 亮（如同鏡射效果）。
- 用 `print()` 在 Serial Monitor 印出目前位置，例如 `pos=2`。
